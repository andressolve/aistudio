Pick something you want to understand better. Then use Claude Code to build yourself a beautiful illustrated guide about it — with AI-generated images included.

What You're Making Today

By the end of this session, you'll have an HTML page you can open in any browser — like a mini-website — that explains your topic with real text and AI-generated illustrations. You're the director. Claude Code is the writer, designer, and illustrator.

The key skill: giving Claude Code a big, complex job and guiding it to a finished product. Not one image. Not one paragraph. A whole thing — start to finish.

Mission 01 — Pick Your Topic
01
Planning
What Do You Want to Learn About?
5 min
›

Before you touch Claude Code, decide what your guide will be about. Pick something you're genuinely curious about — something where you'd actually want to read the final thing.

Here are some ideas. You can use one of these or come up with your own.

⚔️
How Swords Are Made
From raw metal to a finished blade. The forging process, types of steel, why some swords were better than others.
🐸
The Deepest Ocean Creatures
What lives at the bottom of the ocean where no light reaches? How do they survive? What do they look like?
🚀
How SpaceX Lands a Rocket
The engineering behind landing a rocket backwards on a platform in the ocean. Why it was supposed to be impossible.
🧠
How Your Brain Learns New Things
What actually happens inside your head when you practice something. Neurons, myelin, and why repetition works.
🌏
Volcanoes: How They Erupt
Magma chambers, tectonic plates, the biggest eruptions in history, and what makes some volcanoes more dangerous than others.
🤖
How AI Actually Works
Matrix multiplication, training data, neural networks — the real explanation of what's happening when you talk to Claude.
🐙
The Most Dangerous Animals
Not the ones you think. Which animals actually kill the most humans? How do their weapons (venom, strength, speed) work?
🌄
Your Own Idea
Anything you want. History, science, sports, games, animals, technology, mythology — if you're curious about it, it works.
💡
Pick something specific. "Animals" is too broad. "The deepest ocean creatures and how they survive without light" is specific enough to make a great guide. The more focused your topic, the better the result.
  1. Look at the ideas above. Does one of them grab you?
  2. If not, think: what's something I've wondered about recently?
  3. Write down your topic in one sentence. Be specific.

Mission 02 — Direct the Build
02
Creation
Tell Claude Code to Build Your Guide
25 min
›

Open Claude Code. You're going to give it one big, clear instruction — and let it build the whole guide for you. Your job is to be the director: describe what you want, review what it makes, and tell it what to fix.

💬 Say something like this to Claude Code
Create an illustrated guide about [YOUR TOPIC]. Make it as an HTML file I can open in a browser. It should have:

- A cool title and introduction
- 4-5 sections that explain the topic step by step
- An AI-generated image for each section (use Nano Banana to generate them)
- Make it look really good — dark theme, nice fonts, the images should be embedded in the page
- Write it for someone my age who is curious but doesn't know much about the topic yet
Replace [YOUR TOPIC] with what you picked in Mission 1. You can change any of the details — more sections, different style, whatever you want. This is YOUR guide.
🚀
Claude Code will do a LOT of work here. It will write the text, generate image prompts, call Nano Banana to create the images, build the HTML, and put it all together. This might take a few minutes. Let it cook.
🎤
The director's job isn't done after the first request. When Claude Code shows you the result, look at it carefully. Is there a section that's confusing? An image that doesn't match? Something missing? Tell Claude Code specifically what to change. The best guides come from 2-3 rounds of feedback, not from one perfect prompt.
// example conversation — guide about sword forging

You: Create an illustrated guide about how swords are made — from raw metal to a finished blade. Make it as an HTML file. 5 sections, an image for each section using Nano Banana, dark theme, make it look amazing. Write it for someone my age.

Claude Code: [builds the whole thing — writes text, generates 5 images, assembles the HTML]

You (after looking at it): This is cool but the image for the quenching section just shows a generic fire. I want to see the blade being dunked into water with steam rising. Also the section about types of steel is too short — can you add more about the difference between carbon steel and stainless steel?

Claude Code: [regenerates that specific image, expands the steel section]

You: Much better. One more thing — add a section at the end about the most famous swords in history. With images.

Claude Code: [adds the new section with images]

That's 3 rounds. First pass, fix the problems, add something new. Each round the guide gets better because YOU made it better with specific feedback.

  1. Open Claude Code and give it your request.
  2. Wait for the first version. Open the HTML file and read through it.
  3. Find at least ONE thing to improve. Tell Claude Code what to fix.
  4. Review the updated version. Fix one more thing if needed.
  5. When you're happy with it, save it. That's your guide.

Mission 03 — Make It Yours
03
Polish
Add Something Only You Would Think Of
10 min
›

Your guide exists. It works. But right now it's just "a guide about [topic]." This mission turns it into YOUR guide — something that has your personality in it.

Ask Claude Code to add ONE of these (or come up with your own idea):

❔
A Quiz Section
"Add an interactive quiz at the end that tests if the reader actually learned the key facts." Makes the HTML interactive — buttons, answers, scoring.
🗣️
"Myth vs Fact"
"Add a section that busts common myths about this topic." People love being surprised by what they thought was true but isn't.
📈
A Comparison Chart
"Add a visual comparison — like a table or chart — that shows [X vs Y]." Great for topics where you're comparing things.
💭
Your Own Twist
A timeline? A "what would happen if..." section? A rating system? An animated diagram? You decide.
💬 Example
Add an interactive quiz at the end of my guide. 5 questions based on the content. Multiple choice. Show the score at the end. Make it look clean and match the style of the rest of the page.
⭐
This is the director move that matters most. Anyone can ask Claude Code to make a guide. Adding your own creative twist — something unexpected, something that makes it more fun or more useful — is what separates someone who uses AI from someone who directs it.
  1. Pick one addition from above (or invent your own).
  2. Tell Claude Code what to add. Be specific about how it should look and work.
  3. Review the result. Does it fit with the rest of the guide?
  4. Open the final HTML file in your browser. Show it to someone.
